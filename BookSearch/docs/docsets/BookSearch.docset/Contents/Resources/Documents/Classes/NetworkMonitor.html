NetworkMonitor

class NetworkMonitor : ObservableObject

A class responsible for monitoring the network connectivity status using NWPathMonitor.

The NetworkMonitor class observes changes in the device’s network status and publishes the connectivity state (isConnected) that other parts of the app can subscribe to.

Note

The network status is updated on the main thread.
